# What's My IP Address

This script helps you for knowing your IP address.

## Installation

You have to use Debian based distributions for this script.

```bash
git clone https://github.com/rotherda/WhatsMyIP
```

## Usage

```bash
cd WhatsMyIP
./ip.sh
```
