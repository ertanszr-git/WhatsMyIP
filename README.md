# What's My IP Address - WhatsMyIP

This script helps you for knowing your IP address.

## Installation

```bash
git clone https://github.com/ertanszr-git/WhatsMyIP
cd WhatsMyIP
chmod 500 exe.sh
./exe.sh
```

## Usage

```bash
myip
```
